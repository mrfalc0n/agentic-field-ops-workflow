# AppScript Samples

These files are sanitized examples that show the shape of the automation layer. They are not production scripts.

The private implementation used Google Apps Script for:

- Intake validation and promotion.
- Summary refresh logic.
- Spreadsheet protection.
- Backup routines.
- QuickBooks Online OAuth and API glue.

This folder includes small examples only. No credentials, spreadsheet IDs, script IDs, customer data, or production URLs are included.

## Full Controller Sample

`field_ops_controller_sanitized.js` is a full sanitized controller derived from a real Apps Script workflow. It preserves the function structure and operational details while replacing client identifiers, operator names, private document links, and private property keys with generic field-operations terms.
